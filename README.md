# Surveillance Système

Script de surveillance des ressources système.

## Fonctionnalités

### Surveillance CPU
- Affiche l'utilisation actuelle du processeur
- Utilise la commande `top` pour récupérer les statistiques en temps réel

### Surveillance RAM
- Affiche l'utilisation actuelle de la mémoire
- Utilise la commande `free` pour récupérer les statistiques de mémoire
